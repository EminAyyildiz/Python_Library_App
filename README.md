# Python_Library_App
A library application that can add and select books with python


# Bu projedeki temel amacım python üzerinden fonksiyonlar ile birlikte kullanıcı çıkış yapana kadar sürekli bir döngü halinde olan bir kütüphane arayüzü tasarlamaktır. Bu arayüzün kendine has özellikleri vardır. Öncelikle bu arayüz kullancıcıya bir menü sunar ve kullanıcı buradaki menünün işlem numaralarına göre tercihini sisteme bildirir. İlk seçenek ile (1) kullanıcı kütüphaneye istediği gibi kitap ekleyebilir. Bu kitabı eklerken kitabın ismi, kitabın yazarı ve yayınlanma yılını girmek zorundadır. Eğer ki kullanının girmek istediği kitap kütüphanede zaten bulunmaktaysa sistem uyarı vermekte ve kitabı bir daha eklememektedir. İkinci seçenek ile (2) kullanıcı bu kütüphaneden kitap seçebilir. Kullanıcı ismini yazdığı kitabı seçebilmekte ve kullanıcının seçtiği kitap listeden çıkarılmaktadır. Eğer ki bu kitap listede bulunmuyorsa sistem kullanıcıya uyarı vermektedir. Üçüncü (3) seçenek ile ise tüm kütühanedeki kitapların sırasıyla isim,yazar, ve yayım yılına göre listelenip kullanıcıya gösterilmesdir. ve son olarak ise kullanıcı 4  Q  veya q harflerine bastığında uygulamadan çıkış yapabilmektedir. 

#  My main goal in this project is to design a library interface that is in a continuous loop until the user logs out, together with the functions via python. This interface has its own features. First of all, this interface presents a menu to the user, and the user informs the system of his preference according to the operation numbers of the menu here. With the first option (1), the user can add books to the library as he wishes. When adding this book, the name of the book, the author of the book and the year of publication must be entered. If the book that the user wants to enter already exists in the library, the system gives a warning and does not add the book again. With the second option (2) the user can select a book from this library. The user can select the book whose name is written and the book selected by the user is removed from the list. If this book is not in the list, the system gives a warning to the user. With the third (3) option, all the books in the library are listed according to the name, author, and publication year, respectively, and displayed to the user. and finally, the user can exit the application when he presses 4 Q or q letters.

# Mein Hauptziel in diesem Projekt ist es, eine Bibliotheksoberfläche zu entwerfen, die sich in einer Endlosschleife befindet, bis sich der Benutzer zusammen mit den Funktionen über Python abmeldet. Diese Schnittstelle hat ihre eigenen Funktionen. Zunächst präsentiert diese Schnittstelle dem Benutzer ein Menü, und der Benutzer informiert das System über seine Präferenz gemäß den Operationsnummern des Menüs hier. Mit der ersten Option (1) kann der Benutzer Bücher nach Belieben zur Bibliothek hinzufügen. Beim Hinzufügen dieses Buches müssen der Name des Buches, der Autor des Buches und das Erscheinungsjahr eingegeben werden. Wenn das Buch, das der Benutzer eingeben möchte, bereits in der Bibliothek vorhanden ist, gibt das System eine Warnung aus und fügt das Buch nicht erneut hinzu. Mit der zweiten Option (2) kann der Benutzer ein Buch aus dieser Bibliothek auswählen. Der Benutzer kann das Buch auswählen, dessen Name geschrieben ist, und das vom Benutzer ausgewählte Buch wird aus der Liste entfernt. Wenn dieses Buch nicht in der Liste enthalten ist, gibt das System dem Benutzer eine Warnung aus. Mit der dritten (3) Option werden alle Bücher in der Bibliothek nach Name, Autor und Erscheinungsjahr aufgelistet und dem Benutzer angezeigt. und schließlich kann der Benutzer die Anwendung beenden, wenn er 4 Q- oder q-Buchstaben drückt.
